# Project-1---Expenses-Tracker
A simple and responsive expenses tracker web application built using HTML,CSS and Java Script to manage daily income and expenses.
# Features
- Add income and expense transactions
- Automatically calculate total balance
- Display income and expense summary
- Delete transactions
- Responsive user interface

# Technologies Used
- HTML
- CSS
- JavaScript

# Project Purpose
This project was developed to improve my frontend development skills and practice DOM manipulation using JavaScript.

# How It Works
Users can enter income or expense details, and the application updates the total balance dynamically.
